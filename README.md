## Быстрый старт

```bash
# Клонирование и настройка
cd codewithbrain
python -m venv venv
source venv/bin/activate  # Linux/Mac
pip install -r requirements.txt

# База данных
createdb codewithbrain
python manage.py migrate
python manage.py createsuperuser

# Запуск
python manage.py runserver
```
